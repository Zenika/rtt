Comment calculer le nombre de jours de RTT pour une année donnée ?

R + ( ∑ m=1 12 Om ) - T - C + B 12 ∑ m=1 12 Pm ( 1 - Am Om )

R
Le reliquat de l'année précédente. A Zenika, généralement zéro, sauf si vous avez consommé plus de RTT l'année précédente que vous n'en aviez acquis, au quel cas il s'agit du nombre négatif libellé "SOLDE RTT" en bas à gauche de votre bulletin de paie du mois de janvier.
Om
Le nombre de jours ouvrés dans le mois m .
T
Le nombre de jours travaillés défini dans votre contrat de travail. A Zenika, 218.
C
Le nombre de jours de congés payés. A Zenika, 25.
B
Le nombre de jours de RTT bonus. A Zenika, zéro pour les cadres non-autonomes (qui bénéficient de la rémunération des heures supplémentaires), deux pour les cadres autonomes (qui ne bénéficient pas de la rémunération des heures supplémentaires).
Pm
Le taux de temps partiel pour le mois m .
Am
Le nombre de jours d'absences non-rémunérées dans le mois m. Par exemple : les jours de congés sans solde, de carence, de grèves.
